# Hosting on Nest

**TL;DR:** [Nest](https://hackclub.app/) is Hack Club's free Linux VPS (virtual private server). You get your own machine to deploy websites, APIs, bots, databases, or anything else that runs on a server. To start, request an account at [hackclub.app](https://hackclub.app/). This full step-by-step guide is still being written, so for now lean on the official Nest docs and the friendly folks in **#nest** on Slack.

## What you need

* A Hack Club Slack account, so you can request access and ask for help
* A project you want to host (a website, an API, a bot, a database, or anything else that runs on a server)

## What Nest is

[Nest](https://hackclub.app/) is Hack Club's free Linux VPS (virtual private server). You get your own machine to deploy websites, APIs, bots, databases, or anything else that runs on a server. It is a great place to put your project online once it is ready to share.
