## How to use this mirror

- Each `.md` file is a one-to-one copy of a page on the live site.
- Filenames use kebab-case; live slugs are listed alongside each entry.
- If a page changes on the live site, refresh the matching local file.

Format per page below: `Title: <title> | Local: <relative path> | Live: <url>`
